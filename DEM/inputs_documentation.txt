Example:
-----------------------------------------------------------------------
MAX_SPATIAL:				5000
MAX_TEMPORAL:				360
IMAGE_LENGTH:				9440
INTERFEROGRAM_FIRST_X:			100
INTERFEROGRAM_FIRST_Y:			100
X_INTERFEROGRAM_SIZE:			1000
Y_INTERFEROGRAM_SIZE:			1000
COMMONFLAG:				n
ACROSS_LOOKS:				2
DOWN_LOOKS:				2
INTERFEROGRAM_UNWRAPPING_LOOKS:		1

-----------------------------------------------------------------------
Parameter index:description
0:Input the value for the max_spatial parameter for creating sbas_list
1:Input the value for the max_temporal parameter for creating sbas_list
2:Input a value for the length of the image(len_parameter)
3:Input a value for the starting x-coordinate for the interferogram
4:Input a value for the starting y-coordinate for the interferogram
5:Input a value for the size of the interferogram in the x-direction
6:Input a value for the size of the interferogram in the y-direction
7:Input the commonflag parameter(y/n)
8:Input a value for the number of looks across
9:Input a value for the number of looks down
10:Input a value for the number of looks when unwrapping interferograms
